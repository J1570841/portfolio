Proyectos Python

ARTIST

Descripción del proyecto: En Gometrics, participé en el desarrollo de una aplicación de escritorio que, con unos datos de entrada y un excel, generaba un informe para los clientes.

Mi rol: Trabajé de manera individual, utilizando Python.

Tecnologías usadas:

  • Lenguaje: Python
  • Plataforma: Visual Studio Code
  • Colaboración: Git para control de versiones.
  • Bibliotecas usadas:
    • Pandas: Usada para el tratamiento de los distintos excels.
    • NumPy: Usada para tratamiento de los datos de distintos excels.
    • Matplotlib: Usada para dibujar gráficos a partir de los datos tratados de los excels.
    • Pillow (PIL): Usada para convertir imagenes a png.
    • python-docx: Usada para hacer el informe en un word.
    • Tkinter: Usada para crear la inteficie gráfica.
    • tkcalendar: Usada para añadir un calendario en la interficie gráfica.
    • Faker: Usada para las pruebas, para generar datos falsos.
    • unittest: Usada para las pruebas, para generar pruebas unitarias.
    • json: Usada para leer y escribir en archivos json.

Enlace al repositorio: El código está almacenado en un repositorio privado y no puede ser compartido.

Equipo: Trabajo individual.

Bibliotecas usadas

  • Biblioteca para la manipulación y análisis de datos. Proporciona estructuras de datos flexibles y herramientas para limpiar, transformar y visualizar datos.
  • Biblioteca fundamental para la computación científica en Python, que ofrece soporte para matrices y operaciones matemáticas de alto rendimiento.

  • Biblioteca para añadir calendarios a aplicaciones Tkinter, útil para seleccionar fechas de forma visual.
