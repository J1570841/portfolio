Proyectos Web

Fichajes

Descripción del proyecto: En Gometrics, participé en el desarrollo de una plataforma web para fichar las entradas y las salidas de los trabajadores.

Mi rol: Trabajé junto a otro compañero en la implementación del frontend y el backend de la aplicación, utilizando HTML, CSS, JavaScript, PHP y PHPMyAdmin para la base de datos MySQL. Nos encargamos de:

  • Diseñar e implementar la interfaz de usuario (UI), garantizando que fuera intuitiva y fácil de usar.
  • Desarrollar las distintas pantallas web.
  • Creación de un webserver para la posibilidad de hacer una webapp para dispositivos moviles.
  • Realizar pruebas sobre las pantallas y el sistema completo.

Tecnologías utilizadas:

  • Lenguajes: HTML, CSS, JavaScript, PHP
  • Plataforma de base de datos: PHPMyAdmin
  • Colaboración: Git para control de versiones y trabajo en equipo, asegurando la integración continua sin conflictos.
  • Gestión de Tareas: Jira y leantime para la planificación y seguimiento de tareas del proyecto.

Enlace al repositorio: El código está almacenado en un repositorio privado y no puede ser compartido.

Equipo: Trabajo en equipo con otro compañero.

Integración MRP-ERP

Descripción del proyecto: En Gometrics, participé en el análisis y el desarrollo de un nuevo MRP integrado en el ERP de la empresa, para sustituir al MRP que se usaba.

Mi rol: Trabajé junto a otro compañero en el análisi y el desarrollo de este proyecto, utilizando HTML, CSS, JavaScript y PHP. Mi responsabilidad fue:

  • Analizar las funcionalidades del MRP anterior y consultar con los usuarios para identificar sus necesidades y requisitos.
  • Relacionar las tablas de datos del MRP anterior con su funcionalidad, dado que no se tenía acceso al código fuente. Esto es necesario para poder importar los datos al nuevo MRP y que se visualicen correctamente en las pantallas desarrolladas.
  • Desarrollar las distintas pantallas que se integraron en el ERP para el MRP.

Tecnologías utilizadas:

  • Lenguajes: HTML, CSS, javaScript, PHP
  • Plataforma de base de datos: PHPMyAdmin
  • Colaboración: Git para control de versiones y trabajo en equipo.
  • Gestión de Tareas: Jira para la planificación y seguimiento de tareas del proyecto.

Enlace al repositorio: El código está almacenado en un repositorio privado y no puede ser compartido.

Equipo: Trabajo en equipo con otro compañero.
